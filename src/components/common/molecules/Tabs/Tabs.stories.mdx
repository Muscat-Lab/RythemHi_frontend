# Tabs

전반적인 예매 화면에서 사용되어지는 Tabs 컴포넌트입니다.

## Props

```ts
interface TabsProps {
  // tabs 컴포넌트를 이루는 아이템항목들
  tabItems: string[];
  // 현재 클릭된 tab의 idx
  activeIdx: number;
  // tab을 클릭했을때 실행되는 콜백함수
  onClick: () => void;
}
```
