# TitleHeader

프로젝트 내에서 홈, 마이페이지에서 사용되어지는 Title + 특정 페이지로 이동되도록 하는 기능을 포함한 컴포넌트입니다.

## Props

```ts
interface TitleHeaderProps {
  // Title 제목
  title: string;
  // 특정 페이지로 이동하는 콜백함수
  handleGotoRouter: () => void;
}
```
