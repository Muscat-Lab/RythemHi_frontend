# Label

선호하는 아티스트, 장르를 선택한 경우 하단에 보이는 MiniFavorite 컴포넌트 입니다.

( 테두리가 하얀 색이라 흰 배경에서 테두리는 보이지 않습니다! )

## Props

```ts
  text?: string;  // 들어갈 내용
background?: string;  // 선택한 아티스트의 사진이 있을 경우 해당 url주소
favType?: FavType;  // 선택했는지 아닌지 'active' , 'none'이 있다!
```
