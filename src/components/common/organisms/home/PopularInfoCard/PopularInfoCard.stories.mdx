# PopularInfoCard

홈 페이지 내에서 사용되어지는 지금 인기있는 공연 내의 카드 컴포넌트를 표현합니다.

## Props

```ts
// 모두 DB에서 관리되어지는 데이터로, 서버로부터 받아온다.
interface PopularInfoCardProps {
  // 고유 id
  id: string;
  // 현재 카드 index
  currentIdx: number;
  // 카드(공연) 내의 들어갈 이미지 경로
  posterImageUrl: string;
  // 공연 날짜
  performanceDate: string;
  // 공연 제목
  title: string;
  // 장소
  location: string;
}
```
