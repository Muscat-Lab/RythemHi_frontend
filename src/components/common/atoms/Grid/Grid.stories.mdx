# Grid

# Props

```ts
interface GridProps {
  // Grid 행
  rows: number;
  // Grid 열
  columns: number;
  // Grid 행 간격
  rowGap?: number;
  // Grid 열 간격
  columnGap?: number;
  // Grid 안에 들어갈 컴포넌트
  children: React.ReactNode;
}
```
