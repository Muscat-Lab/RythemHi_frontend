# Label

회원가입과 로그인에서 사용되는 인풋 컴포넌트 입니다.

# Props

```ts
interface InputProps {
  // input의 사이즈 단위! 'small', 'medium','large'가 있다.
  inputSize: SizeType;
  // 입력 성공 or 실패에 따라 border-color를 다르게 주기! 'success', 'error'가 있다.
  variant?: VariantType;
}
```
